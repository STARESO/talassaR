# Project TALASSA 

Small repository that contains explanatory scripts for the TALASSA project. Main idea is to be able to explore and analyze the source data given to make the models work. Scripts are here to first explore, then modify datasets. Some also generate error reports or error files that enable the verification of the raw data, in order to correct the original data. Final steps are to generate new processed datasets from the raw datasets, that are usable in the general model. Final result are maps of the pressure effects on habitats. 

Check modelling scripts for everything after the preprocessing steps.
